# Hoc DSA voi LeetCode - 12 Tuan

## Muc tieu
- Nam vung cau truc du lieu va giai thuat co ban den nang cao
- Giai duoc 150+ bai LeetCode
- San sang phong van coding interview

## Cach hoc
1. Moi ngay lam 2-3 bai (1-2 tieng)
2. Doc de bai, suy nghi 15-20 phut truoc khi code
3. Neu khong giai duoc sau 30 phut -> doc goi y (hints)
4. Sau khi AC -> doc Discussion de hoc cach lam tot hon
5. Ghi chu lai pattern/mau giai vao file notes cua tung tuan

## Quy tac dat ten file
- Moi bai lam 1 file: `<so-bai>-<ten-bai>.<ext>`
- Vi du: `001-two-sum.py`, `053-maximum-subarray.js`
- Chon ngon ngu ban thich (Python, Java, C++, JS, ...)
